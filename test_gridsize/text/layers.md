# Número de nós

* Fakenews: Grande queda na penultima camada (5), cai constantemente para valores próximos a 10 nos dois grupos. Pode ser caracteristica do modelo. 
Conforme o grid size aumenta o caimento da curva deixa de ser continuo para virar um grande vale localizado na camada 5. 
O número de nós dessa camada sofre um leve aumento com o o aumento do grid size. Ao atingir o grid size de 50 parece que a curva retorna ao contorno original.

* DPOC: Do grid size de 10 para 20 houve um salto de 20 nós no vale.
Sempre dá a impressão de que a camada 5 esta fugindo do padrão.

* IAM: Apresenta um comportamento diferente do visto no primeiro experimento (para a disciplina) --> Efeito do processo de sanitização, para o primeiro experimentos foram mantidos somente pessoas que responderam as duas perguntas, o que levou a uma redução do conjunto IAM (Os dois datasets ficaram com 430 respostas). No presente experimento só foram removidas respostas em branco ou com conteudo não relacionado, gerando 432 para o DPOC e 447 para o IAM.
Mesmo efeito do vale subir 20 nós quando passa de 10 para 20 no grid size. 
Vale sempre permanece próximo a camada 2.
